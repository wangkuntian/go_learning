# 初始化
按声明顺序初始化。
包由多个.go文件组成，初始化按照编译器收到文件的顺序进行：go工具会在调用编译器前将.go文件进行排序。

```go
func init() { /* ... */ }
```
利用init函数可初始化数组。
init函数不能被调用和引用。当程序启动后，init函数按照声明的顺序自动执行。

包的初始化按照在程序中导入的顺序来进行。初始化过程是自下向上的，main包最后初始化。在程序的main函数执行前，所有的包已经初始化完毕。
